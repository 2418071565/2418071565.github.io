
# 列表初始化

## **基本语法**

在 C 中我们经常见到如下方式的数组初始化方式和结构体初始化方式：

```cpp
int a[4] = { 1, 2, 3, 4 };
struct Point
{
    int x,y;
};
struct Point b = { 1, 2 };
```

这个大括号括起来的列表就被用作初始化数组 `a` 和结构体 `b`，而 C++11 中扩大了该列表的适用范围，使其可以初始化所有内置类型和用户自定义类型，并且在 C++11 中， `=` 也可被省略，如下：

```cpp
// 内置类型
int a[4] { 1, 2, 3, 4 };
double b = { 4.91 };
int c { 4 };

// 用户自定义类型
class Date
{
    int Y,M,D;
public:
    Date(int y,int m,int d):Y(y),M(m),D(d){}
};
Date a { 2024, 4, 6 };
```

对于用户自定义类型，列表初始化会自动调用参数匹配的构造函数。
