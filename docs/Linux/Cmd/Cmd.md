### **ulimit**

`ulimit` 为 shell 内建指令，可用来控制 shell 执行程序的资源。

#### 选项

`-a`：显示所有资源限制。

`-c`：设置core文件的大小限制。

`-d`：设置数据段大小限制。

`-f`：设置文件大小限制。

`-m`：设置内存大小限制。

`-n`：设置打开文件数限制。

`-p`：设置缓冲区大小限制。

`-s`：设置栈大小限制。

`-t`：设置CPU时间限制。
