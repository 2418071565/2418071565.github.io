
## **引入**

之前介绍过文件在内存中是如何被操作系统管理的，但是有很大一部分的文件是在未被打开的，存储在磁盘当中，那么操作系统是如果管理这些磁盘文件的，能让我们快速的寻址访问这些文件数据的呢？

## **文件系统**

对于 Linux 早期使用 ext2 文件系统，后来又发展出 ext3 和 ext4。ext3 和 ext4 虽然对 ext2 进行了增强，但是其核心设计并没有发生变化，所以我们这里只介绍的 ext2。
